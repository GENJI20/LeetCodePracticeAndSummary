# 342. Power of Four

Math Solution.

```java
class Solution {
    public boolean isPowerOfFour(int num) {
        return (Math.log(num) / Math.log(4)) % 1 == 0;
    }
}
```

Recursion. Can be implemented to power of 3, 2 or any other positive numbers.

```java
class Solution {
    public boolean isPowerOfFour(int n) {
        if (n > 1) {
            while (n % 4 == 0) {
                n /= 4;
            }
        }
        return n == 1;
    }
}
```
