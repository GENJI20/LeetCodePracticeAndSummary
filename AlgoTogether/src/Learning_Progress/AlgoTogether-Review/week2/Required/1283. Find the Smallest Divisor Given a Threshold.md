# 1283. Find the Smallest Divisor Given a Threshold

```java
class Solution {
    public int smallestDivisor(int[] nums, int threshold) {
        int left = 0, right = (int)1e6;
        while (left < right) {
            int sum = 0;
            int m = left + (right - left) / 2;
            for (int i : nums) {
                sum += (i + m - 1) / m; // math ceil
            }
            if (sum > threshold) {
                left = m + 1;
            } else {
                right = m;
            }
        }
        return left;
    }
}
```
